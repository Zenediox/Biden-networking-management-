Files included for BYDERN ZENEDIOX website:
- index.html       -> Main HTML file. Links to styles.css and script.js
- styles.css       -> All styles, including smoke animation background
- script.js        -> JavaScript for interactions (scroll, download snapshot)
Instructions:
1. Download the files and keep them in the same folder.
2. Open index.html in a browser to view the site.
3. To change contact info, edit the email/WhatsApp in index.html (elements with id 'emailText' and 'waText').
4. To add product images: on a .thumb element add attribute data-img="images/product1.jpg" and place the image in an images/ folder.
